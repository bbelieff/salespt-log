# MVP 스코프 정의

---

## ⚠️ 중요 원칙

**수강기간은 기수마다 다릅니다. 하드코딩 금지!**
- 수강시작일: 시트 N1에서 읽기
- 계산식: 시작일 + 기간(일)
- 예시 표기: "(예시: PRM 5기의 경우 4/18~6/6)" 형태로만 사용

---

## 용어 정의

| 용어 | 기간 | 계산식 | 용도 |
|---|---|---|---|
| **수강 기간** (Course Period) | 8주 (56일) | 시작일 + 55일 | 주차·퍼널·스킬 점수 집계 |
| **마감 유예** (Follow-up Window) | 2주 (14일) | 수료일 + 14일 | 8주차 말 잡힌 미팅 후속 기록 |
| **편집 가능 기간** (Edit Window) | 10주 (70일) | 시작일 + 69일 | 모든 입력 허용 범위 |
| **완전 종료** (Archived) | 편집 기간 이후 | 시작일 + 70일~ | 읽기 전용, 접근 제한 |

### 동작 규칙

- **주차 네비게이션**: 1~8주 + "📌 마감 유예 (9~10주)" 섹션 분리
- **퍼널/스킬 점수**: 8주 기준 계산 (9~10주 유예 데이터는 별도 배지)
- **미팅 예약**: 편집 기간(10주) 내 어느 날이든 가능
- **시각적 구분**: 1~8주=파란색, 9~10주=앰버(노랑), 11주+=회색(비활성)
- **완전 종료 이후**: 모든 입력 비활성, "수강 완료, 기록 종료" 배너

## MVP 스코프 테이블

### 기간 범위

| 영역 | MVP (IN) | MVP 외 (OUT, 향후) |
|---|---|---|
| **수강 기간** | 8주 (수강시작일 ~ +55일) | 월간/분기/연간 집계 |
| **편집 가능 기간** | 수강 기간 + 마감 유예 2주 (+69일) | 유예 이후 무기한 편집 |
| **통계 집계** | 8주 기준 + 유예 기간 별도 표시 | 장기 트렌드 분석 (3개월, 6개월) |
| **캘린더 네비** | 편집 가능 기간 내만 이동 | 무제한 월/년 이동 |

### 사용자 범위

| 영역 | MVP (IN) | MVP 외 (OUT, 향후) |
|---|---|---|
| **주 사용자** | 현재 수강 중인 수강생 (1인칭 뷰) | 수료생, 일반 영업인 |
| **권한** | 개인 시트 전체 편집 | 다른 수강생 데이터 조회 |
| **협업** | 없음 (개인 기록 전용) | 팀/조직 기능, 공유 |
| **관리** | 자가 관리 | 트레이너/운영자 대시보드 |

### 기능 범위

| 탭 | MVP (IN) | MVP 외 (OUT, 향후) |
|---|---|---|
| **컨택** | 일일 신규/기존 컨택 기록 | 컨택 이력 상세 관리, CRM 연동 |
| **일정** | 미팅 예약/완료 기록 | Google Calendar 동기화 |
| **계약** | 계약/수임비 기록 | 계약서 파일 첨부, 전자서명 |
| **수납** | 입금 기록, 미수금 관리 | 자동 결제, 세금계산서 발급 |
| **대시보드** | 8주 기준 차트 + 유예 집계 | 월간/연간 리포트, AI 분석 |

### 기술 범위

| 영역 | MVP (IN) | MVP 외 (OUT, 향후) |
|---|---|---|
| **인증** | Google OAuth (NextAuth) | 다중 인증, 소셜 로그인 |
| **데이터베이스** | Google Sheets 단일 | 복합 DB, Redis 캐싱 |
| **알림** | 없음 | 푸시 알림, 이메일 알림 |
| **오프라인** | 기본 PWA | 고급 오프라인 동기화 |
| **다국어** | 한국어 전용 | 다국어 지원 |
| **테마** | 기본 테마 | 다크모드, 커스텀 테마 |

## 경계 케이스 처리

### 시간 경계
1. **수강 시작 전 접근**: 읽기 전용, "수강 준비 중" 표시
2. **수강 중 (1~8주)**: 모든 기능 완전 활성
3. **마감 유예 (9~10주)**: 미팅/계약/수납 편집 가능, 주차 배지 "유예" 표시
4. **완전 종료 (11주~)**: 모든 입력 비활성, "수강 완료" 배너

### 데이터 경계
- **8주 범위 내**: 정상 집계, 파란색 표시
- **유예 기간**: 별도 집계, 앰버색 "📌 마감 유예" 배지
- **범위 외**: 표시하지 않음, 회색 비활성

## 확장 가능성 (향후)

이 섹션의 항목들은 MVP에서 의도적으로 제외되었으며, `docs/future/extensions.md`에서 자세히 다룹니다.

### 단기 확장 (P2)
- Google Calendar 동기화
- 푸시 알림 시스템
- 파일 첨부 기능

### 중기 확장 (P3)
- 수료생 구독 서비스
- 트레이너 대시보드
- 월간/분기 리포트

### 장기 확장 (P4)
- 일반 영업인용 SaaS 버전
- 조직/팀 기능
- AI 분석 및 추천
