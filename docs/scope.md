# MVP 스코프 정의 (8주 수강생 전용)

## 🎯 MVP 정의

이번 MVP는 **세일즈피티 현재 수강 중인 8주차 수강생 전용**입니다.

- **기간**: 수강시작일 ~ 수료일 (총 8주, 56일)
- **사용자**: 현재 수강 중인 수강생 (수료생·일반 대중 아님)
- **목표**: 8주 수강 기간 중 영업 활동 기록·점검·성장
- **확장 가능성**: 수료 후 구독 판매, 일반 영업인 대상 확장은 **MVP에서 고려·구현하지 않음**

## 📦 MVP 스코프 테이블

### 기간 범위

| 영역 | MVP (IN) | MVP 외 (OUT, 향후) |
|------|---------|------------------|
| 데이터 기간 | 8주 수강 기간 (56일) | 월간/연간 리포트 |
| 누적 통계 | 8주 누적만 | 무기한 누적 통계 |
| 캘린더 범위 | 수강 시작~수료 월만 | 수강 범위 외 월 이동 |
| 데이터 보존 | 수강 기간 + α | 장기 아카이브 |

### 사용자 범위

| 영역 | MVP (IN) | MVP 외 (OUT, 향후) |
|------|---------|------------------|
| 대상 사용자 | 현재 수강생 1인 | 수료생, 운영자, 일반 영업인 |
| 권한 모델 | 개인 시트 1:1 매핑 | 복잡한 권한 체계 |
| 협업 기능 | 없음 (개인 전용) | 타 수강생 비교·랭킹 |
| 공유 기능 | 없음 | SNS 연동, 초대 기능 |

## 🚧 엣지케이스 처리 (MVP 범위 내)

### 시간 경계 처리
- **수강 시작 전 로그인**: 읽기전용 모드, 데이터 입력 불가
- **수료일 이후 접근**: 모든 탭 읽기전용 + "수료 완료" 배너 표시
- **수강 기간 중**: 정상 편집 가능

### 데이터 범위 제한
- **8주 초과 날짜 진입**: UI에서 차단 또는 "범위 외" 메시지
- **수강 범위 외 월 이동**: 캘린더 네비게이션에서 비활성화

## ⚠️ 스코프 크립 방지 원칙

1. **"언젠가 쓸 수도 있어서"는 금지** - 현재 8주 수강생에게 필요하지 않으면 제외
2. **확장 포인트 미리 준비 금지** - 코드에 미리 구조를 만들지 않음
3. **사용자 요청 필터링** - 스코프 밖 요청은 `docs/future/extensions.md`로 이관
4. **기간 확장 금지** - "한 달만 더 보고 싶다"도 거절

## 🔮 MVP 이후 확장 로드맵

### P1 (즉시 후속) - 수료생 대상
- 수료 후 데이터 조회 (편집 불가)
- 기본적인 구독 결제 모델

### P2 (단기 확장) - 기능 보강
- Google Calendar 동기화
- 푸시 알림
- 데스크톱 반응형

### P3 (중기 확장) - 플랫폼화
- 월간/연간 리포트
- 수료생 커뮤니티
- 타 수강생 비교 기능

### P4 (장기 확장) - SaaS화
- 일반 영업인 대상
- 조직 기능
- 고급 분석·AI
